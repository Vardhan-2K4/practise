Embedding Media in HTML5

1. Embedding Audio (<audio> Tag)

The <audio> element is used to embed sound content in documents.

Attributes used:

  • controls: Displays standard audio controls (play/pause, volume, seek).
  • <source>: Allows you to specify alternative audio files, and the browser will use the first one it supports.
  • type: Specifies the MIME type of the audio file.
Ensure you have 'audio.mp3' (and optionally 'audio.ogg') in the same directory as this HTML file.

2. Embedding Video (<video> Tag)

The <video> element is used to embed video content in documents.

Attributes used:

  • controls: Displays standard video controls (play/pause, volume, seek, full screen).
  • width and height: Sets the dimensions of the video player.
  • preload="metadata": Suggests to the browser to load only metadata (duration, dimensions) to start playback faster.
  • poster: Specifies an image to be displayed before the video starts playing.
  • <source>: Allows you to specify alternative video files and formats for broader browser compatibility.
  • type: Specifies the MIME type of the video file.
Ensure you have 'video.mp4' (and optionally 'video.webm') in the same directory as this HTML file. The `poster` attribute uses a placeholder image.
