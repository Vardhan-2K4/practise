Understanding HTML Lists

1. Unordered List (<ul>)

Items in an unordered list are typically marked with bullet points.

  • Apple
  • Banana
  • Cherry

2. Ordered List (<ol>)

Items in an ordered list are typically marked with numbers or letters.

  1. First Step
  2. Second Step
  3. Third Step

3. Nested Lists

You can embed one type of list inside another.

Unordered List containing an Ordered List:

  • Fruits
    1. Apple
    2. Banana
    3. Orange
  • Vegetables
    1. Carrot
    2. Spinach
    3. Potato

Ordered List containing an Unordered List:

  1. Breakfast Items
    • Cereal
    • Milk
    • Toast
  2. Lunch Items
    • Sandwich
    • Salad
    • Soup

4. Ordered List in an Unordered List

Here's a specific example of an ordered list placed inside an unordered list item.

  • My Favorite Books:
    1. "The Great Gatsby"
    2. "To Kill a Mockingbird"
    3. "1984"
  • My Favorite Movies:
    • Inception
    • The Matrix

5. Definition List (<dl>)

A definition list is a list of terms and their descriptions.

HTML
HyperText Markup Language. The standard markup language for documents designed to be displayed in a web browser.
CSS
Cascading Style Sheets. A style sheet language used for describing the presentation of a document written in HTML.
JavaScript
A programming language that is one of the core technologies of the World Wide Web.
