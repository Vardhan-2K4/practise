Understanding HTML Hyperlinks (<a> Tag)

The <a> tag, also known as the anchor tag, is used to create hyperlinks.

1. The href Attribute

The href (hypertext reference) attribute specifies the URL of the page the link goes to.

2. The target Attribute

The target attribute specifies where to open the linked document.

  • _self: Opens the linked document in the same window/tab where it was clicked (this is the default).
    Example.com (opens in same tab)
  • _blank: Opens the linked document in a new window or tab. This is commonly used for external links.
    W3C (opens in new tab)
  • _parent: Opens the linked document in the parent frame. (Relevant when using frames).
  • _top: Opens the linked document in the full body of the window. (Relevant when using frames).
